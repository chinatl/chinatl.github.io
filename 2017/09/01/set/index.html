  • 具有iterable类型的集合可以通过新的for … of循环来遍历。

小结

  • 看到这，肯定会有强迫症再问我object也是键值对，为什么不用object或者object和map的区别等等，这里给解释一下

    • object通过属性（也就是名字）来访问对应的成绩，而new Map（）构建了一个属于Map类的一个实例对象，这个对象内部有一个map（也就是一系列的键值对），通过对象的get方法访问这个map，从而获取成绩。所以说，两者在实现上完全不同，一个通过属性访问，一个通过方法访问map从而达到目的。

    • 明确职责，object使用hashmap实现，可以把object当map这种数据结构使用，但是object的职责不是map。所以es6专门定义了map，方便使用，职责明确

    • Map的精髓：极快的查找速度
